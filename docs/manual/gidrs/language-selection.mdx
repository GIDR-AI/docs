# Language Selection

{/* AUTO:BEGIN */}


## Overview

Check that the change language functionality is working as expected.

<Info>
This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- At least one team should be created.
- User should be having super admin account.

## Procedure

<Steps>

<Step title="Click the 'Organization' dropdown menu.">

Click the "Organization" dropdown menu.

**Expected result:** The "DropDown" should be clickable, and upon clicking, it should display the list of organizations.

</Step>

<Step title="Select the organization where changes need to be m...">

Select the organization where changes need to be made.

**Expected result:** The "Organizations" should be clickable, and upon clicking, it should display the list of teams.

</Step>

<Step title="Click the 'Team' that requires editing.">

Click the "Team" that requires editing.

**Expected result:** The "Team" should be clickable, and upon clicking, it should open the "Team".

</Step>

<Step title="Click the 'Language' dropdown.">

Click the "Language" dropdown.

**Expected result:** The "dropdown" should be clickable, and upon clicking, it should display the list of options.

</Step>

<Step title="Click the 'Japanese' language option.">

Click the "Japanese" language option.

**Expected result:** The "Japanese" option should be clickable, and upon clicking, it should be reflected in the UI.

</Step>

</Steps>
