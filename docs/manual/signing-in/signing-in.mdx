# Signing In

{/* AUTO:BEGIN */}

## Overview

Verify user is able to log in using valid credentials

<Info>
  This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- A super admin account should be present
- An organization must be created in the US-Central region, and at least one team should be created.
- Organization should be created in US_Central1 region
- User should be having super admin account.
- User should be in landing page

## Procedure

<Steps>
  <Step title="Open the browser and Enter GIDR URL">
    Open the browser and Enter GIDR URL

    **Input:** `https://app.gidr.ai/en`

    **Expected result:** Browser should open and land on "Sign In" page
  </Step>
  <Step title="Click the 'Your email' field and enter user email">
    Click the "Your email" field and enter user email

    **Input:** `vijay.anand.allabothulla@gidr.ai`

    **Expected result:** "Your email" field should be editable
  </Step>
  <Step title="Click the 'Password' field and enter password">
    Click the "Password" field and enter password

    **Input:** `Password1!`

    **Expected result:** "Password" field should be editable
  </Step>
  <Step title="Click the 'SignIn' Button">
    Click the "SignIn" Button

    **Expected result:** The "Sign In" button should be enabled and should respond to clicks.
  </Step>
  <Step title="Verify that the 'Message' toaster popup is display...">
    Verify that the "Message" toaster popup is displayed

    **Expected result:** A success message "Signed in successfully" with a close icon, should appear before the user is redirected to the home page.
  </Step>
</Steps>
