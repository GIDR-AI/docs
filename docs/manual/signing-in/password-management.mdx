# Password Management

## Change password by using forgot password link

Verify whether forgot password link is working as expected

### Prerequisites
User should have valid account but forgot password

### Steps

1. **Open your browser and navigate to the GIDR URL**
   - Data: `https://app.gidr.ai/en`
   - Expected: Verify that the landing page is open

2. **Click on the "Forgot Password" link**
   - Expected: Verify that the page navigates to the 'Forgot Password' screen

3. **Click on the "Your email" field**
   - Expected: The "Your email" field should be clickable and upon clicking it should allow user to inputs.

4. **Enter your email**
   - Expected: Verify that the field allows user inputs and displays them.

5. **Click on the "Reset password" Button**
   - Expected: The "Reset password" button should be clickable.

6. **Verify that the button is clickable**
   - Expected: The "Button" should be clickable and , upon clicking it has to navigate to the "Sign-In" screen.

7. **Navigate to the next screen**
   - Expected: Validate that the message 'We've sent you an email. Check your email and follow the instructions to reset your password.' is displayed.
