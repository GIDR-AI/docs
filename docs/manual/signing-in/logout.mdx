# Logout

{/* AUTO:BEGIN */}

## Verification and validation for Logout functionality.

Verify that the logout functionality is working as expected.

### Prerequisites
User should be in the Home screen with all data loaded.

### Steps

1. **Verify that the "Home" screen is loaded**
   - Expected: The "Home" screen should be loaded with all the modules and the side navigation bar.

2. **Click on the "Account details" button.**
   - Expected: The "Account Details" button should be clickable, and upon clicking, it should open the "Member Profile" and display the "Logout" button, which should also be clickable.

3. **Click on the "Logout" button**
   - Expected: The "Logout" button should be clickable, and upon clicking, it should navigate back to the "Sign-in" screen.

## Verification and validation for super admin logout functionality.

Check whether the super admin logout flow is working as expected

### Prerequisites
User should be having super admin account.

### Steps

1. **User as to login to the account with valid credentials and landed to the dash board.**
   - Expected: It should land on the Dashboard with all UI elements loaded.

2. **Click on the "Profile" Image icon.**
   - Expected: The "Icon" should be clickable, and upon clicking it, all the options should be displayed.

3. **Click on the "Logout" button.**
   - Expected: The "Logout" button should be clickable, and upon clicking, it should log out the user and navigate back to the sign-in screen.
