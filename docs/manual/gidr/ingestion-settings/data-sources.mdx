# Data Sources

## Overview

Verify and validate data module is working as expected

<Info>
This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- User should be created at least one GIDR and inside the GIDR preview Page.
- User should be in GIDR page with all data to be loaded

## Procedure

<Steps>

<Step title="Click the 'Data' menu from sidebar">

Click the "Data" menu from sidebar

**Expected result:** The 'Data Source' popup should open with all data fully loaded.

</Step>

<Step title="Click the 'Add' button">

Click the "Add" button

**Expected result:** The popup should display with the header (Connect your data), a close icon, radio buttons (Files, URLs, Integrations), a click-to-upload button, and "Back" and "Next" buttons. Also that the "Files" radio button is selected by default.

</Step>

<Step title="Click the 'Click to upload' button">

Click the "Click to upload" button

**Expected result:** The "Click to upload" container should be clickable. Upon clicking, it should open the File explorer window.

</Step>

<Step title="Select the File from the File explorer">

Select the File from the File explorer

**Expected result:** Choose a file to upload, and ensure the uploaded file is displayed.

</Step>

<Step title="Click the 'Next ' button">

Click the "Next " button

**Expected result:** The "Next" button should be clickable. Upon clicking, it should redirect to GIDR page.

</Step>

<Step title="Verify that the 'Toaster' message is displayed aft...">

Verify that the "Toaster" message is displayed after the file upload.

**Expected result:** It should display a success message stating "File uploaded".

</Step>

<Step title="Click the 'Data source' popup close icon">

Click the "Data source" popup close icon

**Expected result:** It should be clickable, and upon clicking, it should close the "Data Source" popup.

</Step>

</Steps>
