# Gidr Info

{/* AUTO:BEGIN */}


## Overview

Verify editing of GIDR is working as expected

<Info>
This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- User should be created at least one GIDR

## Procedure

<Steps>

<Step title="Check whether the teams page is displayed.">

Check whether the teams page is displayed.

**Expected result:** The team's page should load with all list of created GIDR's.

</Step>

<Step title="Click the 'GIDR' on which you wants to edit.">

Click the "GIDR" on which you wants to edit.

**Expected result:** The GIDR which is selected should be displayed and navigated to the GIDR page.

</Step>

<Step title="Click the 'GIDR info'">

Click the "GIDR info"

**Expected result:** The "GIDR info" popup contains the created information, indicates the published status, and displays placeholders for "GIDR Name" and "GIDR Description," as well as a Logo field with a "Change" button and a "Save Changes" button.

</Step>

<Step title="Click the 'GIDR name'' field to edit and update th...">

Click the "GIDR name"' field to edit and update the GIDR name.

**Expected result:** The "GIDR name" field should be editable

</Step>

<Step title="Click the 'Description' field to edit and update t...">

Click the "Description" field to edit and update the GIDR Description.

**Expected result:** The "Description" field should be editable

</Step>

<Step title="Check and Verify that the default logo is displaye...">

Check and Verify that the default logo is displayed and "Change" button is working as expected

**Expected result:** The default logo should be displayed and "Change" button should be clickable ,and upon clicking it should navigate to the file explore.

</Step>

<Step title="Click the 'Change' button to select a new logo.">

Click the 'Change' button to select a new logo.

**Expected result:** The 'Change' button should be clickable, and upon clicking, it should display the ''Change logo'' popup, which includes a "Click to Upload'' button.

</Step>

<Step title="Click the 'Click to Upload' button to select and u...">

Click the 'Click to Upload' button to select and upload a file.

**Expected result:** The "Click to upload" container should be clickable. Upon clicking, it should open the File explorer window.

</Step>

<Step title="Select the File from the File explorer">

Select the File from the File explorer

**Expected result:** Choose a file to upload, and ensure the uploaded file is displayed.

</Step>

<Step title="Click the 'Save' button">

Click the "Save" button

**Expected result:** Ensure that the uploaded file is displayed.

</Step>

</Steps>
