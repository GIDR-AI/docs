## Prerequisites

- User should be in GIDR page

## Procedure

<Steps>

<Step title="Click the 'Prompt'.">

Click the "Prompt".

**Expected result:** It should display "Prompt" popup.

</Step>

<Step title="Click the 'Edit prompt'">

Click the "Edit prompt"

**Expected result:** The "Prompt" popup should show a prompt message, an "Edit Prompt" button, and a close icon.

</Step>

<Step title="Click the 'Edit prompt' button.">

Click the "Edit prompt" button.

**Expected result:** The "Edit prompt" button should be clickable, and upon clicking as to enable prompt message field.

</Step>

<Step title="To enter a new message, Click the 'Prompt input fi...">

To enter a new message, Click the "Prompt input field" and type desired text.

**Expected result:** The "Prompt Input" field should be editable.

</Step>

<Step title="Click the 'Save' button">

Click the "Save" button

**Expected result:** The "Save" button should be clickable, and upon clicking, it should store the most recent entries.

</Step>

<Step title="Click the 'Prompt close' icon">

Click the "Prompt close" icon

**Expected result:** The "Prompt close" icon should be clickable, and upon clicking it should close the Prompt popup.

</Step>

</Steps>
