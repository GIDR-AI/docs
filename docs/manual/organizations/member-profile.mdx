# Member Profile

## Overview

Verify that the member profile functionalities is working as expected.

<Info>
This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- User should be having a account.
- User should be in "Home" screen.

## Procedure

<Steps>

<Step title="Click the &quot;Account details&quot; button">

Click the &quot;Account details&quot; button

</Step>

<Step title="Verify that the &quot;Member Profile&quot; is disp...">

Verify that the &quot;Member Profile&quot; is displayed

**Expected result:** Verify that the &quot;Logout&quot; button is clickable

</Step>

<Step title="Click the &quot;Member Profile&quot; button">

Click the &quot;Member Profile&quot; button

**Expected result:** Verify that the &quot;Member Profile&quot; screen is loaded with all data

</Step>

<Step title="Verify that the &quot;Member Details&quot; section...">

Verify that the &quot;Member Details&quot; section is displayed

**Expected result:** Are the &quot;First Name&quot;, &quot;Last Name&quot;, and &quot;Email&quot; fields clickable and editable?

</Step>

<Step title="Click the &quot;First Name&quot; field">

Click the &quot;First Name&quot; field

</Step>

<Step title="Enter a new first name">

Enter a new first name

**Expected result:** Verify that the &quot;First Name&quot; field allows user inputs and reflects the changes

</Step>

<Step title="Click the &quot;Last Name&quot; field">

Click the &quot;Last Name&quot; field

</Step>

<Step title="Enter a new last name">

Enter a new last name

**Expected result:** Verify that the &quot;Last Name&quot; field allows user inputs and reflects the changes

</Step>

<Step title="Verify that the &quot; Save changes&quot; button i...">

Verify that the &quot; Save changes&quot; button is enabled and Clickable

</Step>

<Step title="Verify that the &quot;Success&quot; toaster is dis...">

Verify that the &quot;Success&quot; toaster is displayed

**Expected result:** Verify that the page displays the message &quot;Member updated successfully&quot;

</Step>

</Steps>

## Parameters

The following parameters are used in this process:

- `Language: English`
- `Language: Japanese`
- `Success Message: "Organization updated successfully"`
