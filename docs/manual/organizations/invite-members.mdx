# Invite Members

{/* AUTO:BEGIN */}

## Overview

Check that resend invite is working as expected.

<Info>
This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- At least one team should be created
- User should be having organization access.

## Procedure

<Steps>

<Step title="Click the 'Organization' dropdown menu.">

Click the "Organization" dropdown menu.

**Expected result:** The "DropDown" should be clickable, and upon clicking, it should display the list of organizations.

</Step>

<Step title="Select the organization where changes need to be m...">

Select the organization where changes need to be made.

**Expected result:** The "Organizations" should be clickable, and upon clicking, it should display the list of teams.

</Step>

<Step title="Click the 'Account Details' button.">

Click the "Account Details" button.

**Expected result:** The "Account Details" Button should be clickable, and upon clicking, it should display the list of options.

</Step>

<Step title="Click the 'Organization settings' option.">

Click the "Organization settings" option.

**Expected result:** The "Organization settings" button should be clickable, and when clicked, it should navigate to the "Organization Details" screen.

</Step>

<Step title="Click the 'Members' module.">

Click the "Members" module.

**Expected result:** The "Members" module should be clickable, and upon clicking, it should open the member screen.

</Step>

<Step title="Click the 'Action/3 Dots' button.">

Click the "Action/3 Dots" button.

**Expected result:** The "Action/3 Dots" button should be clickable, and upon clicking, it should display the list of options within it.

</Step>

<Step title="Click the 'Resend invite' button.">

Click the "Resend invite" button.

**Expected result:** The "Resend invite" button should be clickable, and upon clicking, it should open the "Resend Invite" popup with the "Resend Invite to Teams" dropdown.

</Step>

<Step title="Click the 'Add to teams' button">

Click the "Add to teams" button

**Expected result:** The "Add to teams" should be clickable, and upon clicking, it should display a success toast message stating, "Invitation has been sent successfully".

</Step>

</Steps>
