# Organization Settings

{/* AUTO:BEGIN */}


## Overview

Ensure that the organization settings are functioning as expected.

<Info>
This feature allows you to accomplish the task described above. Follow the procedures below to get started.
</Info>

## Prerequisites

- The user should have access to the organization, and at least one organization should be created.

## Procedure

<Steps>

<Step title="Click the 'Organization' dropdown menu.">

Click the 'Organization' dropdown menu.

**Expected result:** The "DropDown" should be clickable, and upon clicking, it should display the list of organizations.

</Step>

<Step title="Select the organization where changes need to be m...">

Select the organization where changes need to be made.

**Expected result:** The "Organizations" should be clickable, and upon clicking, it should display the list of teams.

</Step>

<Step title="Click the 'Account Details' button.">

Click the "Account Details" button.

**Expected result:** The "Account Details" Button should be clickable, and upon clicking, it should display the list of options.

</Step>

<Step title="Click the 'Organization settings' option.">

Click the "Organization settings" option.

**Expected result:** The "Organization settings" button should be clickable, and when clicked, it should navigate to the "Organization Details" screen.

</Step>

<Step title="Update the 'Organization Name' with a new one.">

Update the "Organization Name" with a new one.

**Expected result:** The "Organization name" should be clickable and editable, allowing for the name to be updated with a new one.

</Step>

<Step title="Change the 'Default Language' to Japanese.">

Change the "Default Language" to Japanese.

**Expected result:** The "Default Language" Dropdown should be clickable, and upon clicking, it should display the entire list of options.

</Step>

<Step title="Click the  'Japanese' Language option.">

Click the  "Japanese" Language option.

**Expected result:** The "Japanese" option should be clickable, and selecting it should update the UI language to Japanese.

</Step>

<Step title="Click the 'Save change' Button.">

Click the "Save change" Button.

**Expected result:** The "Save changes" button should be clickable, and upon clicking, it should save the changes made.

</Step>

</Steps>
